BodyStyle

Libreria de estilos CSS para la rapida creacion de la interfaz de usuario UI

Comenzar GitHub

Con Bodystyle se puede crear sitios responsive sin demasiado esfuerzo y con mucha ganancia de tiempo en el desarrollo de aplicaciones web.

Diseños adaptables a todos los tamaños de dispositivos con ganancia de tiempo y una estética de los modelos más elevada.
En el presente sitio se muestran todas las funcionalidades de la librería junto con la lógica que acompaña a cada uno de los componentes.

Vamos a ver los diferentes métodos de descarga e instalacion de bodystyle

Podemos agregar la librería a través del enlace que figura en la parte inferior a este texto en la cabecera del documento principal del proyecto en desarrollo.

Es necesario para utilizar las funcionalidades de la librería disponer del archivo bodystyle.min.js el cual podemos agregarlo a través de su CDN que figura en la parte inferior a este texto.

Si querremos disponer de los archivos físicos de la librería podemos descargar el archivo pre-compilado desde el botón que figura en la parte inferiror a este texto.

 Descargar

A través de la cuenta de GitHub que figura en la parte inferior a este texto podemos acceder a todo el código fuente de bodystyle.

 Código Fuente

Podemos instalar Bodystyle con NPM a través de la consola con el comando que figura debajo.

 npm install body-ui

También podemos instalar Bodystyle a través de la consola utilizando Yarn con el comando que figura debajo.

 yarn add body-ui
